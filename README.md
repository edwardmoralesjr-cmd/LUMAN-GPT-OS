# LUMAN GPT OS

LUMAN GPT OS is Edward Morales Jr.'s command center for building, refining, testing, and versioning custom GPTs.

This repository is the source-of-truth for reusable GPT instructions, project-specific knowledge templates, tone rules, test prompts, changelogs, and future action/API specifications.

## Core Mission

Turn Edward's GPTs from isolated prompts into living systems with:

- Clear operating instructions
- Shared voice and style rules
- Project-specific knowledge files
- Version history
- Test prompts
- Changelogs
- Upgrade paths

## Golden Rule

This repo stores GPT design, templates, canon, workflows, and instructions. It should not store private secrets, passwords, encryption keys, API keys, seed files, live financial balances, employer-confidential data, or anything that would cause harm if exposed.

## LUMAN Principle

The system should serve clarity, peace, creativity, discipline, family stability, spiritual depth, and practical execution.
