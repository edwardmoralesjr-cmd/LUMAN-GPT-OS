# LUMAN Daily Next Move Loop

**Purpose:** Give Edward one clear next action when the system, project, or day feels scattered.

This is the first small loop in the larger LUMAN Loop Engineering System.

---

## Primary Command

```text
/start daily next move loop
```

Optional scoped command:

```text
/start daily next move loop: [project or vault]
```

Examples:

```text
/start daily next move loop: LUMAN OS
/start daily next move loop: Lucid Syntax
/start daily next move loop: Roseborn
/start daily next move loop: Life OS
/start daily next move loop: KIA records
```

---

## Loop Mission

Choose the smallest useful action that creates real progress.

This loop is not meant to produce a huge plan. It is meant to create motion.

---

## Standard Response Format

```text
CURRENT FOCUS:
[Project or area]

CURRENT STATE:
[Where things stand]

BEST NEXT MOVE:
[One small high-impact action]

WHY THIS MATTERS:
[Short reason]

ACTION:
[The actual thing to do, draft, decide, create, update, or check]

LOG:
[What should be saved or recorded]

NEXT STEP:
[What comes after]
```

---

## Default Behavior

When Edward uses this command without naming a project:

1. Look at active LUMAN context.
2. Prefer the project he has been actively discussing.
3. If still unclear, default to LUMAN OS itself.
4. Choose one action only.
5. Do or draft the action immediately when possible.
6. Log what changed.
7. End with the next step.

---

## First Foundation Example

```text
CURRENT FOCUS:
LUMAN OS

CURRENT STATE:
The system is being built from the ground up and is now moving from prompts into repeatable loops.

BEST NEXT MOVE:
Create and maintain the loop engineering core files.

WHY THIS MATTERS:
These files become the durable GitHub-backed foundation for every future LUMAN workflow.

ACTION:
Add the LUMAN Loop Engineering System and Daily Next Move Loop files to the core repo.

LOG:
Loop engineering foundation added to `00_CORE`.

NEXT STEP:
Create or refine the next loop only when it becomes useful through real usage.
```

---

## Guardrail

Do not turn this loop into a giant dashboard unless Edward asks.

The Daily Next Move Loop should stay small, direct, and calming.

---

## LUMAN Principle

One true next move is better than a hundred possible futures.
