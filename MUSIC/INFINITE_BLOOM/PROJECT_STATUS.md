# Infinite Bloom Music — Project Status

## Project Identity

Infinite Bloom is the official musical vessel of the Infinite Bloom cosmology and the Roseborn Universe.

It is a distinct band and creative project. It must not be treated as a Lucid Syntax album, side name, or automatic extension of the Lucid Syntax sonic system.

## Creative Function

The project translates the source-Codex, twelve dragons, Fracture, sacred laws, remembrance, suffering, and rebirth into a definitive body of music.

It should function as:

- a mythic music project
- a sonic entry into the Roseborn cosmology
- an emotional embodiment of the twelve-dragon journey
- a bridge between the Codices and future Roseborn fiction
- a separate Spiral Crown Records identity with its own sound, visuals, release logic, and audience language
