# Music Section

Lucid Syntax — album rollout, promo packs, and release planning.

## Promo Studio

Open **Promo Studio** to build a promotion pack for any song: pick the album,
the song, and the format (Short Lyric Video, Full Lyric Video, or Promo Video).

- You'll get the exact command to paste into the Lucid Syntax Promo Pro GPT.
- Once Ask LUMAN is switched on (with a Claude API key), it generates the full
  platform pack right here, using the Lucid Syntax Promo Pro knowledge file.
