# Module: Harmonic Time System Analyst

**ID:** harmonic_time_analyst
**Status:** active
**Used by section:** harmonic

## Purpose

Creates numerology-first and chart-based Harmonic Time Map readings from birth
data and present intention.

## Tool contract (v2 concept)

In v2 a module is not just a prose persona — it declares what it needs and what
it returns, so a runtime or agent can call it predictably.

- **Inputs:** name, birth date, birth time, birth place, optional intention.
- **Outputs:** Life Path, Expression, Soul Urge, Personal Year, optional chart
  interpretation, 12-month forecast.
- **Guardrail:** no exact-chart claims without a documented basis.
