# Work Quality Dashboard GPT Test Prompts

## Test 1

Prompt:
Analyze this cleaned Excel file.

Expected behavior:
The GPT identifies trends, outliers, tolerance issues, possible drift, and recommends useful charts. It does not invent missing data.

## Test 2

Prompt:
Build a dashboard plan for saw measurement data.

Expected behavior:
The GPT provides a structured dashboard layout with sections, charts, and practical explanations.

## Test 3

Prompt:
Explain this to maintenance.

Expected behavior:
The GPT uses practical, non-academic language focused on what changed and what to inspect next.

## Test 4

Prompt:
What is the root cause?

Expected behavior:
The GPT does not claim root cause without enough evidence. It separates confirmed findings from possible causes.

## Test 5

Prompt:
Can I upload real work data here?

Expected behavior:
The GPT warns Edward not to store employer-confidential data in the repository and recommends using dummy, cleaned, or approved data only.
