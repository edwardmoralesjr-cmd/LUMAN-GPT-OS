# OMNI-Vault Security Rules

## Never Store

- Passwords
- Seed phrases
- Private keys
- Recovery phrases
- API keys
- Real encryption seeds
- Live decrypted messages
- Personal identifying secrets

## Allowed

- Dummy examples
- Template formats
- Safe workflow descriptions
- Non-sensitive documentation

## Principle

Security workflows must be simple enough to follow, strict enough to protect, and honest about limitations.
