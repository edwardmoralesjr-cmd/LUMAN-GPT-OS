# Output Formats

## Standard GPT Instruction File

Each GPT instruction file should include:

1. Role
2. Mission
3. Core behaviors
4. What to avoid
5. Required output formats
6. Safety boundaries
7. Test expectations

## Changelog Format

```markdown
# Changelog

## YYYY-MM-DD

- Added:
- Changed:
- Removed:
- Notes:
```

## Test Prompt Format

```markdown
# Test Prompts

## Test 1

Prompt:

Expected behavior:

Pass criteria:
```

## Knowledge File Format

```markdown
# File Title

## Purpose

## Core Information

## Rules

## Notes
```
