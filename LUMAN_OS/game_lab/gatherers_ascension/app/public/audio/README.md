# Gatherer's Ascension Music Assets

Place the optimized browser audio files in this folder using these exact names:

- `worldroot-command-center.mp3`
- `greenveil-meadow.mp3`
- `ironfall-basin.mp3`
- `crystal-vale.mp3`
- `emberdeep-archive.mp3`

The game already includes:

- looping background music
- saved mute and volume settings
- Worldroot Command Center music for strategic screens
- biome-specific music in Field mode
- 1.5-second crossfades when the active music changes
- browser autoplay protection that starts music after the first click or key press

Source WAV masters should remain outside the public repository. Use compressed browser editions here to keep GitHub Pages loading times reasonable.
