# Gatherer's Ascension Roadmap

## Milestone 1: Collection Loop v1

Status: Implemented and build validated

- Hidden Mythic variant for every current material
- Unknown silhouettes before first harvest
- Full Collection Codex tab
- Material lore, properties, uses, rarity, and native regions
- Standard, Fine, Perfected, Ancient, and Enchanted qualities
- Discovery reveal overlay
- Critical harvests and surprise events
- Discovery momentum and temporary resonance boosts
- Collection milestones
- Avatar and tool visual progression foundations

## Milestone 2: Sensory Reward Pass

- Unique gathering sounds by tool and material family
- Quality-tier audio stingers
- Mythic discovery music sting
- Particle effects and material-specific animations
- Camera and screen effects tuned by rarity
- Haptic feedback for supported mobile devices
- Accessibility controls for flashes, shake, sound, and motion

## Milestone 3: Tool Infusion and Build Identity

- Infuse tools with discovered Mythic materials
- Branching tool traits
- Named tool forms
- Visible model and effect changes
- Tool loadouts
- Non-destructive respec system
- Collection-set synergies

## Milestone 4: Deeper Avatar Progression

- Character creation
- Unlockable clothing and gathering outfits
- Aura and trail cosmetics
- Titles earned through collections and mastery
- Region-specific visual adaptations
- Special gathering abilities

## Milestone 5: Expanded Gathering World

- Fishing
- Excavation
- Salvaging
- Tracking
- Weather and time-dependent resources
- Hidden micro-biomes
- Traveling rare-resource events
- Environmental puzzles

## Milestone 6: Long-Term Collection Systems

- Regional collection pages
- Material-family mastery
- Museum or personal archive display
- Collection completion cosmetics
- Duplicate Mythic-material uses
- Masterwork crafting
- Prestige that expands options without deleting progress

## Recommended Next Move

Complete Milestone 2 with a focused sensory-reward pass before expanding the number of biomes.
