# Gatherer's Ascension UI-First Command System

## Locked Direction

Gatherer's Ascension is now a premium UI-first gathering command RPG. The command interface is the primary game surface. Direct field gathering remains important, but it now operates inside a larger management game about building a world-spanning gathering network.

## Core Dual Loop

```text
Direct Field Loop
Explore -> Gather -> Discover -> Upgrade -> Enter deeper locations

Command Network Loop
Recruit -> Equip -> Deploy -> Automate -> Collect -> Expand -> Optimize
```

Both loops feed the same inventory, Codex, avatar level, tool progression, biome access, rare discovery system, and long-term economy.

## Command Interface Law

The player should feel like the operator of a living resource-intelligence network. The interface should present useful information through readable gauges, progress bars, maps, operation queues, activity feeds, collection tracking, assignment timers, equipment telemetry, and world-state readouts.

The UI is not decorative. It is the primary way the player understands, manages, and improves the game.

## Major Command Views

### Dashboard

The strategic home screen. It summarizes the current biome, active deployments, inventory value, collection progress, network rank, gatherer capacity, rare-discovery potential, objectives, and recent operation results.

### Field

The direct Phaser gathering mode. The player moves through the active biome, harvests nodes, discovers materials, and triggers high-value moments while automated gatherers continue operating in the background.

### Network

The world and deployment map. It organizes the world into:

```text
World -> Biomes -> Regions -> Gathering Zones -> Resource Pools
```

Players unlock zones, assign one specialist per active zone, monitor repeating extraction cycles, and expand deeper into the world.

### Gatherers

The roster and loadout command screen. Gatherers have individual levels, equipment tiers, efficiency, endurance, fortune, specialties, traits, assignments, completed cycles, and lifetime yield.

### Codex

The full collection-intelligence archive. Direct and automated discoveries share the same permanent Codex.

### Upgrades

The investment screen for player tools, field equipment, gatherer loadouts, and future network infrastructure.

## Automated Gathering Rules

- The first gatherer is available immediately so automation is part of the game from the beginning.
- Additional gatherers unlock through player level, command capacity, and recruitment costs.
- Gatherers operate in repeating cycles until recalled.
- Automation continues while the player is in another screen or away from the game.
- Offline progress is calculated from elapsed time and capped to protect balance and performance.
- Zone duration is modified by gatherer efficiency, specialty match, equipment tier, and network level.
- Automated gathering can discover new base resources, elevated qualities, and Mythic variants.
- Every automated result advances the gatherer, player, network, inventory, Codex, and lifetime statistics.
- Direct play should remain the strongest source of active surprise events and visual spectacle.

## Progression Layers

### Player

Level, XP, permanent stats, tool mastery, field gear, command rank, and biome access.

### Gatherer

Individual level, XP, specialty, equipment tier, efficiency, endurance, fortune, assignment history, and lifetime yield.

### Network

Network level, network XP, command points, gatherer slots, unlocked zones, simultaneous operations, and completed cycles.

### World

Biome access, zone access, regional stability, danger, resource families, rarity modifiers, and expedition depth.
