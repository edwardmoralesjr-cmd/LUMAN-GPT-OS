## Portfolio Rules

### Rule 1: Three active strategic fronts maximum

The default strategic fronts are:

```text
[1] Human foundation and family peace
[2] One shipping creative project
[3] One deep-building creative or system project
```

Maintenance systems may continue, but they should not become additional strategic fronts unless a real problem demands it.

### Rule 2: Every project needs a next gate

A project is not actively managed until it has:

- A project home
- A current state
- A source owner
- A next gate
- A clear definition of done for that gate

### Rule 3: Finish before revealing the deepest layer

The Infinite Bloom and the widest Roseborn cosmology should remain sequenced behind enough completed music, Codex refinement, and grounded fiction to support the reveal.

### Rule 4: Contradictions stop downstream expansion

When a major canon or architecture contradiction exists, log and reconcile it before generating dependent outlines as though the conflict were settled.

### Rule 5: Public repository safety always wins

Private financial, medical, family, employer, credential, and exact personal records must remain outside this public repository.
