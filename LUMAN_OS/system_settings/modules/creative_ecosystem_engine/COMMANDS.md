# Creative Ecosystem Engine Commands

## `/open creative ecosystem`

Open this module's manifest, current health view, commands, and next integration gate.

## `/synthesize`

Find meaningful connections among current material and the wider creative ecosystem. Return suggested relationships only. Do not silently make them canonical.

## `/pivot`

Recommend a related task that provides active recovery while preserving thematic, strategic, or dependency value.

## `/evolve`

Compare proposed work with earlier material and classify it as evolution, refinement, repetition risk, or disconnected work.

## `/recycle`

Find dormant, archived, completed, or published material with unrealized transformation value.

## `/gaps`

Identify missing musical, literary, visual, practical, philosophical, or worldbuilding counterparts.

## `/next`

Rank next actions by priority, momentum, cross-project value, dependency value, emotional alignment, completion probability, anti-stagnation value, time fit, and burnout risk.

## `/lineage`

Trace where a concept originated and how it has transformed across projects.

## `/ecosystem`

Show ecosystem health, project balance, synthesis opportunities, gaps, dormant nutrients, and stagnation risks.

## `/transform`

Propose ways to move an existing concept into another medium or project form.

## `/stagnation-check`

Detect blocked work, repeated themes without evolution, excessive active-node load, neglected high-value material, and integration imbalance.

## Required Ending

Every command response should end with one `Recommended Next Move` when appropriate.
