# Creative Ecosystem Engine

This folder contains the first implementation-ready base for LUMAN OS cross-project synthesis and anti-stagnation logic.

## Included

- System manifest and governance
- Command reference
- Framework-neutral TypeScript engine
- Example data and command execution
- Update specification
- Integration checklist

## Install locally

```bash
npm install
npm run check
npm run build
```

## Boundary

This code is a base engine, not a live autonomous writer. Permanent relationships, canon changes, and project activation still require the appropriate LUMAN OS governance and Edward's approval.
