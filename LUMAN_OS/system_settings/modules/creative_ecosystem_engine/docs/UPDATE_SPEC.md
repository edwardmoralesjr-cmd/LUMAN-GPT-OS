# Creative Ecosystem Engine Update Specification

## Foundational doctrine

Edward's creative projects form one living ecosystem. Music, books, Codices, Roseborn mythology, philosophy, visual systems, family learning, games, and lived experience should be connected without flattening their distinct purposes.

## Required behaviors

### Synthesis

Identify meaningful relationships using themes, symbols, emotions, lineage, dependencies, and project context.

### Context pivot

When work is blocked or energy is mismatched, recommend lateral movement that continues the larger mission. A context pivot counts as active recovery only when it preserves coherent value.

### Anti-stagnation

Monitor blocked active work, dormant high-value material, repeated themes, project overload, excessive organization without creation, creation without integration, and unused transformation potential.

### Evolution

New work should deepen, reinterpret, challenge, embody, or transform earlier work. Similarity alone does not equal development.

### Gap detection

Detect philosophy without music, songs without documented philosophy, doctrine without character embodiment, recurring symbols without definitions, themes without visual language, and lived lessons without practical form.

### Next best action

Rank work using project priority, momentum, cross-project value, dependency value, emotional alignment, completion probability, anti-stagnation value, available-time fit, and burnout risk.
