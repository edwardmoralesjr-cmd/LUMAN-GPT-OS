# LUMAN OS Book Catalog Commands

## Primary Command

```text
/open book catalog
```

Open:

```text
LUMAN_OS/book_section/BOOK_IDEA_MASTER_CATALOG.md
```

## Commands

### `/book registry`

Open:

```text
LUMAN_OS/book_section/BOOK_PROJECT_REGISTRY.md
```

### `/book folder: [project]`

Open the matching folder under `BOOKS/` and show the recovered-history file plus any mature project sources.

### `/book recovery status`

Open:

```text
LUMAN_OS/book_section/BOOK_HISTORY_RECOVERY_STATUS.md
```

### `/book missing sources`

Open:

```text
LUMAN_OS/book_section/MISSING_SOURCE_RECOVERY_QUEUE.md
```

### `/classify book: [project]`

Classify as:

```text
Published
Active Building
Active Drafting
Blocked
Incubating
Companion
Derivative
Concept
Archived
```

### `/activate book: [project]`

Before activation, verify:

- Source owner
- Current state
- Next gate
- Definition of done
- Dependencies
- Capacity within the active-front limit

### `/recover book history: [project]`

Search accessible GitHub sources, retained conversation history, uploaded files, and user-provided old transcripts for additional material. Append only new information and preserve provenance.

### `/merge book overlap: [projects]`

Compare overlapping concepts and recommend:

- Keep separate
- Parent / child relationship
- Companion relationship
- Merge
- Archive one version

## Required Response Ending

```text
Recommended Next Move
```

## Recommended Next Move

```text
Open the missing-source queue and recover the highest-value unavailable source file.
```
