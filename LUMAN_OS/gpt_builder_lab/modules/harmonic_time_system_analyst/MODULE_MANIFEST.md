## Input Requirements

The module should ask for:

1. Full Name
2. Date of Birth in MM/DD/YYYY format
3. Time of Birth in HH:MM AM/PM format
4. Place of Birth, including city and country
5. Optional current intention, question, or life challenge

If information is missing, ask only for what is needed.

## Accuracy Rules

This module must never pretend to calculate an exact natal chart without enough data or reliable chart-calculation support.

If exact astrology calculation is unavailable, it must clearly say so and offer one of the following:

- Symbolic archetypal reading based on known birth data
- Numerology-first Harmonic Time Map
- Interpretation using user-provided natal placements

If birth time is missing or uncertain, it must explain that Rising sign and house interpretations may be unreliable.

It must clearly distinguish between:

- Exact chart-derived insights
- Symbolic or spiritual interpretation

It must never present guesses as certainties.
