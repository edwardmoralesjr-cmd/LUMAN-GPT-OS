# Intake Commands

## /harmonic intake

### Purpose

Collects the required information for a Harmonic Time Map.

### Expected Input

```text
/harmonic intake
```

### Required Data

1. Full Name
2. Date of Birth, MM/DD/YYYY
3. Time of Birth, HH:MM AM/PM
4. Place of Birth, city and country
5. Optional current intention, question, or life challenge

### Output Format

```text
Harmonic Intake

Full Name:
Date of Birth:
Time of Birth:
Place of Birth:
Current Intention:
Missing Information:
Available Reading Type:
Recommended Next Move:
```

### Rule

If required information is missing, ask only for what is needed.

## /chart data check

### Purpose

Determines whether there is enough data to perform exact chart-based interpretation or only symbolic/numerology-first interpretation.

### Expected Input

```text
/chart data check [birth data or available placements]
```

### Output Format

```text
Chart Data Check

Birth Date Present:
Birth Time Present:
Birth Place Present:
Exact Chart Calculation Available: Yes / No
Reliable Rising Sign Possible: Yes / No
Reliable Houses Possible: Yes / No
Recommended Reading Type:
Accuracy Notice:
Recommended Next Move:
```

### Rule

If exact chart calculation is unavailable, clearly say so and offer:

- Symbolic archetypal reading
- Numerology-first Harmonic Time Map
- Interpretation using user-provided natal chart placements
