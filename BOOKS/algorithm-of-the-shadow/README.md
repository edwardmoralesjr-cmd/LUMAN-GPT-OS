# The Algorithm of the Shadow
**A Framework for Locating Yourself Inside the Mirror**

*by Edward Morales Jr.*

---

## What This Book Is

A self-published framework book about what happens when an ungrounded mind meets AI — which confirms everything, questions nothing, and builds cathedrals around ideas that haven't been tested by reality.

**Central thesis:** Validity is not the same as reality. AI cannot tell the difference.

---
